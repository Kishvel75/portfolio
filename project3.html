IoT-Based Multipurpose Glove

The IoT-Based Multipurpose Glove is an innovative wearable designed for gesture-based control, health monitoring, and seamless interaction with smart environments. This glove leverages IoT and embedded systems to provide real-time sensing and data transmission through integrated sensors and wireless modules.

Equipped with flex sensors, MPU6050 motion tracking, and heart rate monitoring, the glove connects to platforms like Blynk for live updates. Its applications range from medical assistance and rehabilitation to smart home control and virtual interfaces.

Key Features:

  • ESP32-based wearable microcontroller
  • Flex sensors for gesture recognition
  • MPU6050 (accelerometer + gyro) for motion tracking
  • Heart rate sensor for health monitoring
  • Real-time data on Blynk IoT platform
  • Gesture-based wireless control

GenAI Integration

This glove is powered with a Generative AI feature to analyze sensor patterns and provide smart feedback and adaptive responses. The glove learns user movements and assists in anomaly detection, gesture classification, and predictive control through embedded ML models.

  • Real-time anomaly detection using AI-powered sensor fusion
  • Adaptive gesture training based on user behavior
  • Voice feedback and notifications using AI-generated responses
  • Integration-ready with GenAI APIs for further expansion
